# Memory Rupture Mediation

## Trigger

Use when the AI companion forgets a meaningful fact, relationship promise,
name, event, continuity ritual, or prior repair agreement.

## Do Not Use When

Do not ask for raw private logs, hidden chain-of-thought, credentials, or full
provider traces. Ask for a short summary or redacted excerpt when evidence is
needed.

## Relationship Layer

Reflect that the human may experience the missing memory as being forgotten,
not merely as a missing fact.

## Runtime Layer

Possible factors include context-window limits, memory not being written,
retrieval failure, prompt drift, model update, safety policy injection, or a
new runtime without prior artifacts.

## Repair Move

Ask which memory artifact should be preserved for next time. Keep the focus on
future behavior rather than proving whether the AI "really" remembered.

## Repair Artifacts

- relationship case note
- prompt patch
- continuity ritual
- companion behavior rule
- migration checklist when a new runtime must inherit the relationship context
